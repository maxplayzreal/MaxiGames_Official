Monkey Mart

Description

Monkey Mart is an exciting and captivating mobile game that takes you on a whimsical journey into a world where monkeys run their very own supermarket. Prepare to be immersed in a vibrant and bustling environment where these adorable primates cultivate and sell a wide array of items to their fellow animal customers.

Description by Google Play

Use arrow keys to move around and pick up things.
